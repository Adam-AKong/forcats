Change the levels of a factor

Usage

fct_recode(f, ...)

Arguments

f
A factor.
...
A sequence of named character vectors where the name gives the new level, and the value gives the old level. Levels not otherwise mentioned will be left as is. Levels can be removed by naming them NULL.

Description

Change the levels of a factor

Examples

x <- factor(c("apple", "bear", "banana", "dear"))
fct_recode(x, fruit = "apple", fruit = "banana")


[1] fruit bear  fruit dear 
Levels: fruit bear dear




# If you make a mistake you'll get a warning
fct_recode(x, fruit = "apple", fruit = "bananana")

Warning message:
Unknown levels in `f`: bananana

[1] fruit  bear   banana dear  
Levels: fruit banana bear dear




# If you name the level NULL it will be removed
fct_recode(x, NULL = "apple", fruit = "banana")


[1] <NA>  bear  fruit dear 
Levels: fruit bear dear
