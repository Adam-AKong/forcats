Count entries in a factor

Usage

fct_count(f, sort = FALSE)

Arguments

f
A factor
sort
If TRUE, sort the result so that most common values float to the top.

Value

A tibble with columns f and n

Description

Count entries in a factor

Examples

f <- factor(sample(letters)[rpois(1000, 10)])
table(f)
