Anonymise factor levels

Usage

fct_anon(f, prefix = "")

Arguments

f
A factor
prefix
A character prefix to insert in front of the random labels.

Description

Replaces factor levels with arbitary numeric identifiers. Neither the values nor the order of the levels are preserved.
